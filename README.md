# zh-ai-flavor-refiner

**祛机润文：中文 AI 味诊断与文本修订 Skill。**

用于识别和修订中文文本中的机械感、模板化句壳、空泛判断、过度整齐、语体错配，以及外部检测反馈标记的高风险片段。

本项目不承诺规避任何检测系统，也不以制造错别字、语病或随机口语化来模拟“真人感”。它的核心目标是：在保留事实、术语、引用、数字、语体和作者意图的前提下，减少不必要的机器味。

## 适用场景

- 中文文章、报告、论文、工作材料的机械感诊断。
- 对已有文本做局部修订，让表达更具体、更贴合场景。
- 根据外部检测报告或标记片段，定位并处理需要人工回读的位置。
- 区分“正文正在使用套话”和“规则、词表、报告正在讨论套话”，减少元文本误伤。

## 两种模式

- **quick（默认）**：大多数场景。不加载 reference，只扫篇章/句壳/词汇三层，aggressive 强度，直接输出。适用于公众号、社交文本、工作消息。
- **full**：论文、公文、法律、技术文档、带检测报告、用户明确要求审校。加载 reference，8 层诊断，standard 强度，保真回读。

## 使用方式

把整个目录复制到支持 Skill 的 tools 目录。触发词包括"去模板味""说人话""humanize""机器味太重"等。

## 隐私规则

公开仓库只应包含通用规则、合成示例和占位符。请勿提交真实语料、私人项目、客户材料、检测报告截图、知识库日志、云盘路径和任何可识别个人身份的信息。
